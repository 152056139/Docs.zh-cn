# <a name="enforcing-ssl-in-an-aspnet-core-app"></a>在 ASP.NET Core 应用程序实施 SSL

作者：[Rick Anderson](https://twitter.com/RickAndMSFT)

本文档说明如何：

- 要求将 SSL 用于所有请求 （仅适用于 HTTPS 请求）。
- 所有 HTTP 请求重都定向到 HTTPS。
