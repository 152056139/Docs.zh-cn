## <a name="what-are-partial-views"></a><span data-ttu-id="d24bc-107">什么是分部视图？</span><span class="sxs-lookup"><span data-stu-id="d24bc-107">What are Partial Views?</span></span>

<span data-ttu-id="d24bc-108">分部视图是指在其他视图内呈现的视图。</span><span class="sxs-lookup"><span data-stu-id="d24bc-108">A partial view is a view that's rendered within another view.</span></span> <span data-ttu-id="d24bc-109">通过执行分部视图生成的 HTML 输出在调用视图（或父视图）中呈现。</span><span class="sxs-lookup"><span data-stu-id="d24bc-109">The HTML output generated by executing the partial view is rendered into the calling (or parent) view.</span></span> <span data-ttu-id="d24bc-110">和视图一样，分部视图也使用 *.cshtml* 文件扩展名。</span><span class="sxs-lookup"><span data-stu-id="d24bc-110">Like views, partial views use the *.cshtml* file extension.</span></span>

## <a name="when-should-i-use-partial-views"></a><span data-ttu-id="d24bc-111">应在何时使用分部视图？</span><span class="sxs-lookup"><span data-stu-id="d24bc-111">When Should I Use Partial Views?</span></span>

<span data-ttu-id="d24bc-112">分部视图是将大型视图分解为较小组件的有效方法。</span><span class="sxs-lookup"><span data-stu-id="d24bc-112">Partial views are an effective way of breaking up large views into smaller components.</span></span> <span data-ttu-id="d24bc-113">它们可减少视图内容的重复并使视图元素得以重复使用。</span><span class="sxs-lookup"><span data-stu-id="d24bc-113">They can reduce duplication of view content and allow view elements to be reused.</span></span> <span data-ttu-id="d24bc-114">常见布局元素应在 [_Layout.cshtml](layout.md) 中指定。</span><span class="sxs-lookup"><span data-stu-id="d24bc-114">Common layout elements should be specified in [_Layout.cshtml](layout.md).</span></span> <span data-ttu-id="d24bc-115">非布局可重用内容可封装到分部视图中。</span><span class="sxs-lookup"><span data-stu-id="d24bc-115">Non-layout reusable content can be encapsulated into partial views.</span></span>

<span data-ttu-id="d24bc-116">如果拥有由多个逻辑部分组成的复杂页面，将每个部分用作其自己的分部视图十分有用。</span><span class="sxs-lookup"><span data-stu-id="d24bc-116">If you have a complex page made up of several logical pieces, it can be helpful to work with each piece as its own partial view.</span></span> <span data-ttu-id="d24bc-117">页面的每个部分都可独立于页面的其余部分进行查看，且页面本身的视图也会变得更加简单，因为它仅包含整体页面结构以及用于呈现分部视图的调用。</span><span class="sxs-lookup"><span data-stu-id="d24bc-117">Each piece of the page can be viewed in isolation from the rest of the page, and the view for the page itself becomes much simpler since it only contains the overall page structure and calls to render the partial views.</span></span>

<span data-ttu-id="d24bc-118">提示：在视图中遵守[不要自我重复原则](http://deviq.com/don-t-repeat-yourself/)。</span><span class="sxs-lookup"><span data-stu-id="d24bc-118">Tip: Follow the [Don't Repeat Yourself Principle](http://deviq.com/don-t-repeat-yourself/) in your views.</span></span>

## <a name="declaring-partial-views"></a><span data-ttu-id="d24bc-119">声明分部视图</span><span class="sxs-lookup"><span data-stu-id="d24bc-119">Declaring Partial Views</span></span>

<span data-ttu-id="d24bc-120">分部视图的创建方式与任何其他视图的创建方式类似：在 *Views* 文件夹内创建 *.cshtml* 文件。</span><span class="sxs-lookup"><span data-stu-id="d24bc-120">Partial views are created like any other view: you create a *.cshtml* file within the *Views* folder.</span></span> <span data-ttu-id="d24bc-121">分部视图和常规视图之间没有语义差异-，仅呈现方式不同。</span><span class="sxs-lookup"><span data-stu-id="d24bc-121">There's no semantic difference between a partial view and a regular view - they're just rendered differently.</span></span> <span data-ttu-id="d24bc-122">可拥有直接从控制器的 `ViewResult` 返回的视图，并可将同一视图用作分部视图。</span><span class="sxs-lookup"><span data-stu-id="d24bc-122">You can have a view that's returned directly from a controller's `ViewResult`, and the same view can be used as a partial view.</span></span> <span data-ttu-id="d24bc-123">视图和分部视图的主要呈现方式差异在于分部视图不运行 *_ViewStart.cshtml*（而视图运行 — 有关 *_ViewStart.cshtml* 的详细信息，请参阅[布局](layout.md)）。</span><span class="sxs-lookup"><span data-stu-id="d24bc-123">The main difference between how a view and a partial view are rendered is that partial views don't run *_ViewStart.cshtml* (while views do - learn more about *_ViewStart.cshtml* in [Layout](layout.md)).</span></span>

### <a name="partial-view-discovery"></a><span data-ttu-id="d24bc-134">分部视图发现</span><span class="sxs-lookup"><span data-stu-id="d24bc-134">Partial View Discovery</span></span>

<span data-ttu-id="d24bc-135">引用分部视图时，可通过多种方式引用其位置：</span><span class="sxs-lookup"><span data-stu-id="d24bc-135">When referencing a partial view, you can refer to its location in several ways:</span></span>

```text
// Uses a view in current folder with this name
// If none is found, searches the Shared folder
@Html.Partial("ViewName")

// A view with this name must be in the same folder
@Html.Partial("ViewName.cshtml")

// Locate the view based on the application root
// Paths that start with "/" or "~/" refer to the application root
@Html.Partial("~/Views/Folder/ViewName.cshtml")
@Html.Partial("/Views/Folder/ViewName.cshtml")

// Locate the view using relative paths
@Html.Partial("../Account/LoginPartial.cshtml")
```

<span data-ttu-id="d24bc-136">可在不同视图文件夹中拥有具有相同名称的不同分部视图。</span><span class="sxs-lookup"><span data-stu-id="d24bc-136">You can have different partial views with the same name in different view folders.</span></span> <span data-ttu-id="d24bc-137">按名称（不带文件扩展名）引用视图时，每个文件夹中的视图都会使用与其位于同一文件夹中的分部视图。</span><span class="sxs-lookup"><span data-stu-id="d24bc-137">When referencing the views by name (without file extension), views in each folder will use the partial view in the same folder with them.</span></span> <span data-ttu-id="d24bc-138">还可指定要使用的默认分部视图，将其放在 *Shared* 文件夹中。</span><span class="sxs-lookup"><span data-stu-id="d24bc-138">You can also specify a default partial view to use, placing it in the *Shared* folder.</span></span> <span data-ttu-id="d24bc-139">任何没有属于自己的分部视图的视图则可使用共享分部视图。</span><span class="sxs-lookup"><span data-stu-id="d24bc-139">The shared partial view will be used by any views that don't have their own version of the partial view.</span></span> <span data-ttu-id="d24bc-140">可设置默认分部视图（位于 *Shared* 中），该视图被与父视图位于同一文件夹并具有相同名称的分部视图替代。</span><span class="sxs-lookup"><span data-stu-id="d24bc-140">You can have a default partial view (in *Shared*), which is overridden by a partial view with the same name in the same folder as the parent view.</span></span>

<span data-ttu-id="d24bc-141">分部视图可*链接*。</span><span class="sxs-lookup"><span data-stu-id="d24bc-141">Partial views can be *chained*.</span></span> <span data-ttu-id="d24bc-142">也就是说，分部视图可调用其他分部视图（只要未创建循环）。</span><span class="sxs-lookup"><span data-stu-id="d24bc-142">That is, a partial view can call another partial view (as long as you don't create a loop).</span></span> <span data-ttu-id="d24bc-143">在每个视图或分部视图内，相对路径始终相对于该视图，而不相对于根视图或父视图。</span><span class="sxs-lookup"><span data-stu-id="d24bc-143">Within each view or partial view, relative paths are always relative to that view, not the root or parent view.</span></span>
