### <a name="cloud-slas-compared-to-enterprise-down-time-experience"></a><span data-ttu-id="38ef1-178">云服务级别协议相比企业停机时间体验</span><span class="sxs-lookup"><span data-stu-id="38ef1-178">Cloud SLAs compared to enterprise down-time experience</span></span>

### <a name="not-all-downtime-counts-toward-slas"></a><span data-ttu-id="38ef1-198">并非所有停机时间将计入 Sla</span><span class="sxs-lookup"><span data-stu-id="38ef1-198">Not all downtime counts toward SLAs</span></span>
